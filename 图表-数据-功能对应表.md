# 数据可视化平台 - 图表-数据-功能对应表

## 项目概述
本项目是一个包含10类图表的高级数据可视化平台，基于ECharts 5.4.3 + Tailwind CSS构建，支持丰富的交互功能和响应式设计。

## 图表配置清单

### ① 折线图 (Line Chart)
- **数据来源**: 未来15天气温数据
- **数据结构**: 
  - X轴：15天日期序列
  - 系列1：最高温度（随机生成 20-35°C）
  - 系列2：最低温度（随机生成 10-20°C）
- **交互功能**: 
  - 悬浮显示详细温度数据
  - 平滑曲线展示
  - 支持缩放平移
  - 实时数据更新

### ② 柱形图 (Bar Chart)
- **数据来源**: 淘宝天猫GMV数据
- **数据结构**: 
  - X轴：Q1-Q4季度
  - 系列：GMV数值（亿元）
  - 误差棒：±10-15%范围
- **交互功能**: 
  - 显示最大/最小值标记
  - 误差范围可视化
  - 季度数据对比

### ③ 堆积条形图 (Stacked Bar Chart)
- **数据来源**: 商品网购替代率
- **数据结构**: 
  - Y轴：商品类别（服装、电子产品、日用品等）
  - 系列1：线上购买比例
  - 系列2：线下购买比例
- **交互功能**: 
  - 堆积显示总比例
  - 分类对比分析
  - 百分比显示

### ④ 堆积面积图 (Area Chart)
- **数据来源**: 物流公司费用分布
- **数据结构**: 
  - X轴：物流公司（顺丰、中通、圆通等）
  - 系列1：运输费用
  - 系列2：仓储费用
  - 系列3：其他费用
- **交互功能**: 
  - 面积堆积显示
  - 费用构成分析
  - 公司间对比

### ⑤ 直方图 (Histogram)
- **数据来源**: 人脸识别灰度值
- **数据结构**: 
  - 数据：100个随机灰度值（0-255）
  - 分组：20个等距区间
- **交互功能**: 
  - 灰度分布统计
  - 频数显示
  - 分布密度分析

### ⑥ 饼图/圆环图 (Pie/Donut Chart)
- **数据来源**: 支付宝月账单
- **数据结构**: 
  - 支出类别：餐饮美食、交通出行、购物消费等
  - 金额数据：模拟消费金额
- **交互功能**: 
  - 圆环图显示
  - 百分比标签
  - 点击突出显示
  - 类别筛选

### ⑦ 散点图/气泡图 (Scatter/Bubble Chart)
- **数据来源**: 汽车速度与制动距离
- **数据结构**: 
  - X轴：速度（30-150 km/h）
  - Y轴：制动距离（10-60 m）
  - 气泡大小：随机尺寸
- **交互功能**: 
  - 气泡大小映射
  - 平均值参考线
  - 数据点悬浮详情

### ⑧ 箱形图 (Box Plot)
- **数据来源**: 2017-2018全国发电量
- **数据结构**: 
  - X轴：地区（华北、华东、华南等）
  - 数据：各地区发电量样本
- **交互功能**: 
  - 五数概括显示
  - 离群点检测
  - 地区间对比

### ⑨ 雷达图 (Radar Chart)
- **数据来源**: 霍兰德职业兴趣测试
- **数据结构**: 
  - 维度：现实型、研究型、艺术型等6个
  - 数据：个人测评分数
- **交互功能**: 
  - 多边形面积显示
  - 多维度评估
  - 标准化评分

### ⑩ 误差棒图 (Error Bar Chart)
- **数据来源**: 4个树种细根生物量
- **数据结构**: 
  - X轴：树种（橡树、松树、枫树、桦树）
  - 数据：平均值 + 误差范围
- **交互功能**: 
  - 误差范围可视化
  - 平均值对比
  - 统计显著性显示

## 功能特性说明

### 高级交互功能
1. **加载动画**：所有图表支持平滑加载动画
2. **悬浮交互**：鼠标悬浮显示详细数据
3. **主题切换**：支持5种配色方案
4. **数据间距**：可自定义数据点间距
5. **CSV下载**：支持数据导出为CSV格式
6. **实时修改**：支持实时数据编辑和更新
7. **缩放平移**：支持图表缩放和平移操作

### 设计布局特点
1. **响应式设计**：适配电脑/平板/手机设备
2. **顶部导航**：图表全览下拉菜单
3. **视图切换**：单图表/多图表分栏显示
4. **底部说明**：图表对应实例标注
5. **扁平化界面**：简洁大气的设计风格

### 技术实现
1. **技术栈**：HTML/CSS/JavaScript + ECharts + Tailwind CSS
2. **性能优化**：加载速度≤3秒
3. **兼容性**：现代浏览器全面支持
4. **部署简单**：无需复杂部署，打开HTML即可运行

## 使用说明

### 基本操作
1. **选择图表**：使用顶部下拉菜单切换不同图表类型
2. **视图切换**：点击"单图表"或"多图表"按钮切换显示模式
3. **主题切换**：点击颜色按钮切换界面主题

### 数据操作
1. **编辑数据**：点击"编辑数据"按钮打开数据编辑模态框
2. **实时更新**：点击"实时更新"按钮启动/停止数据实时更新
3. **下载CSV**：点击"下载CSV"导出当前图表数据
4. **重置缩放**：点击"重置缩放"恢复图表原始视图

### 交互操作
1. **悬浮查看**：鼠标悬浮在图表上查看详细数据
2. **缩放平移**：使用鼠标滚轮缩放，拖动图表平移
3. **图例控制**：点击图例可显示/隐藏对应数据系列
